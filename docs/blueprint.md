# **App Name**: CraftCanvas Portfolio

## Core Features:

- Project Showcase: Display selected projects with images, descriptions, and links.
- About Section: An about me section detailing relevant experience and skills.
- Contact Form: Enable visitors to send messages through a simple form.
- AI-Powered Portfolio Enhancement Tool: An AI tool that helps users to write more engaging project descriptions.

## Style Guidelines:

- Primary color: Soft terracotta (#E07A5F) to convey warmth and creativity.
- Background color: Off-white (#FAF9F6), providing a clean and subtle backdrop.
- Accent color: Deep teal (#3D405B) to offer contrast and sophistication.
- Headline font: 'Playfair', serif, to evoke elegance and professionalism. Body font: 'PT Sans', sans-serif, for readability and modernity.
- Minimalist, line-based icons for navigation and key sections, ensuring clarity without overwhelming the user.
- Grid-based layout ensuring content is well-organized, balanced, and responsive across devices.
- Subtle transition animations to enhance user experience and create a sense of refinement, such as fading elements and smooth scrolling.
